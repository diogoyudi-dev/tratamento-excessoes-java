# Trabalhando com Excessões

## Sobre o Projeto

Projeto desenvolvido com o objetivo de praticar conceitos de tratamento de exceções em Java.

O sistema simula uma conta bancária simples, permitindo a realização de saques e validando regras de negócio através de exceções personalizadas.


## Objetivos de Aprendizagem
* Criar exceções personalizadas.
* Utilizar throw para lançar exceções.
* Utilizar try-catch.
* Aplicar herança em exceções.
* Validar regras de negócio.
* Organizar classes em pacotes.


## Funcionalidades
* Criação de conta bancária.
* Validação de saldo inicial.
* Validação de limite.
* Realização de saque.
* Tratamento de erros com mensagens específicas.


## Regras de Negócio
* O sistema lança exceções quando:
* O saldo inicial é negativo.
* O limite informado é inválido.
* O valor de saque é menor ou igual a zero.
* O valor de saque ultrapassa o saldo disponível.
* O valor de saque ultrapassa o limite permitido.


## Tecnologias Utilizadas
* Java
* Programação Orientada a Objetos (POO)
* Tratamento de Exceções
